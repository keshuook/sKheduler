sKhedule

About

sKhedule is a simple app which helps you keep track of things. It does so by allowing you to create alarms.

Help

To view your alarms and edit them, you must go to the dashboard page by either clicking on the button labled "Dashboard" on the navigation bar on the top or by going to the page http://localhost:[port]/dashboard. To create an alarm, go to the create page by either clicking the button labled "Create" on the navigation bar or by going to the page http://localhost:[port]/create.
